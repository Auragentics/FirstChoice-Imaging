🎯 Overview: Option B Multi-Intent Approach

How It Works:

  • Voice agent handles multiple intents in a single call (Records, Billing, Lien)
  • Each intent uses prefixed variables to prevent overwrites
  • {{intents_handled}} tracks which workflows were completed
  • Webhook automation parses {{intents_handled}} to determine which emails to send

Key Variable: {{intents_handled}}

  • Comma-separated string of completed workflows
  • Possible values: "records", "billing", "lien"
  • Example: "records,billing" = caller addressed both intents

🔄 Master Automation Flow

Result:

  • 1-3 emails sent based on {{intents_handled}} (records, billing, lien)
  • Each email uses the corresponding prefixed variables
  • SMS confirmation sent to caller

Variable Description
